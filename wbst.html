What is HTML?

HTML (HyperText Markup Language) is the standard language used to create web pages
and web applications. It provides the basic structure of a webpage, which can then be
enhanced and modified using CSS (Cascading Style Sheets) and JavaScript. HTML
consists of a series of elements or tags that define different parts of the content, such as
headings, paragraphs, links, images, and more. Each element is enclosed in angle brackets
(e.g., <tagname> content </tagname>). The purpose of HTML is to ensure that web content
is structured in a meaningful and accessible way for both users and web browsers.

Basic Structure of HTML


This is the Basic structure of an HTML webpage: 

   
                <!DOCTYPE html> 
                <html>
                    <head>
                        <title> Title goes here! </title>
                    </head>
                    <body>
                        <h1> This is heading 1. We can add up to 6 headings in HTML! </h1>
                        <p> Here goes Paragraph! Use "<br>" tag to add new lines in paragraphs! </p>
                    </body>
                </html>

What is CSS?

Cascading Style Sheets (CSS) is a styling language used to describe the presentation of a
web page, defining how HTML elements should be displayed. It enables developers to apply styles
such as colors, fonts, spacing, and layout to web content, providing a consistent and visually appealing
design across multiple pages. CSS allows for separation of content from design, making it easier to maintain
and update the look of a site. It also supports responsive design techniques, ensuring that web pages adapt to
different screen sizes and devices. CSS styles can be applied inline, internally within the HTML, or externally
through linked stylesheets, offering flexibility in how styles are managed and reused. You can add CSS into an
HTML document using any of 3 ways.
They are as follows:
(1) Inline CSS: This means to add CSS directly into an HTML element using the style attribute.
(2)Internal CSS: This means Adding CSS code in the HTML Document using the <style> tag in the <head> section.
(3)External CSS: This means keeping CSS code in an external file and linked to the HTML
document using the <link rel="stylesheet" type="text/css" href="filename.css" > tag.

Example of Inline CSS

This text is 10 pixels and is in red color!

To get something similar like this, this is the following inline css code used to attain it: 

 <h6 style="font-size: 10px; color:red;" > This text is 10 pixels only! </h6>
You may also use h1, h2, h3, h4 or h5 instead of h6 here!

Example of Internal CSS

This text is 20 pixels and is in orange color!

To get something similar like this, this is the following internal css code used to attain it: 

 
                Head Section:
                <style>
                h5 {
                font-size: 20px;
                color: rgb(255,165,0);
                }
                </style>
                
                Body Section: 
                <h5> This text is 20 pixels and is in orange color! </h5>
You may also use h1, h2, h3, h4 or h6 instead of h5 here!

Example of External CSS

This text is in Script font-style, 30 pixel font-size and white font-color!

To get something similar like this, this is the following external css code used to attain it: 

                Head Section in HTML file:
                <link rel="stylesheet" type="text/css" href="filename.css" > 

                Body Section in HTML file:
                <h4> This text is in Script font-style, 30 pixel font-size and white font-color! </h4>

                CSS file:
                @import url('https://fonts.googleapis.com/css2?family=Dancing+Script&display=swap'); 
                //The above code is used to get the cursive (or script) font from google fonts!
                h4 {
                    font-family: 'Dancing Script', cursive;
                    font-style: italics;
                    font-size: 30px;
                    color: White;
                }

What is JavaScript?

JavaScript (JS) is a versatile, high-level programming language commonly used in web
development to create dynamic and interactive user experiences. It allows developers to
manipulate HTML and CSS, handle events, and perform complex computations on the web page.
JavaScript operates on the client side (in the browser) as well as on the server side (with Node.js),
making it a powerful tool for both front-end and back-end development. Its key features include
event-driven programming, asynchronous operations (using callbacks, promises, and async/await),
and a rich set of built-in objects and libraries that facilitate tasks ranging from DOM (Document Object Model)
manipulation to data visualization.
You can add JS to an HTML document using the following ways:
(1) Inline JS: Inline JS means to add JS code directly within the HTML document, typically within specific HTML tags or attributes.
(2) Internal JS: Internal JS means placing JS code within the HTML document using the <script> tag in the <head> or <body> section.
(3) External JS: External JS means placing JS code in an external file with a .js extension and linked to the HTML document using <script> tag.

Example for Inline JS

The following code is used to create this alert box and button: 

                <button onclick="alert('This alert is made through Inline JS!')"> Click me! </button>

Example for Internal JS

The following code is used to create this alert (with button and function): 

 
                Button code:
                <button onclick="Button()" > Click me to start the example of Internal JS! </button>

                Body section:
                <script>
                function Button () {
                alert ("This alert box is created through Internal JS!");
                }
                </script>

Example of External JS

The following code is used to create this alert (with button and function): 

                Button code:
                <button onclick="Activate()" > Click me to start the example of External JS! </button>

                Linking the JS file:
                <script src="filename.js" > </script>

                JS file:
                function Activate () {
                alert ("This alert box is created through External JS!");
                }
